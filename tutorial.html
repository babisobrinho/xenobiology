Tutorial do Jogo

Bem-vindo ao Laboratório de Xenobiology!

Neste laboratório virtual, você assumirá o papel de um cientista estudando bactérias alienígenas com genética desconhecida. Prepare-se para uma jornada de descobertas científicas!

Dica: Este tutorial contém informações que seu cientista ainda não conhece no jogo. Use sabiamente para orientar sua pesquisa!

Objetivo do Jogo

Seu objetivo principal é:

  • Descobrir como os genes R, K e W interagem
  • Entender suas hierarquias de dominância
  • Criar uma colônia resistente a bacteriófagos mutantes
  • Publicar descobertas científicas baseadas em seus experimentos

Você começará sem conhecer as regras genéticas - elas serão reveladas através de experimentação cuidadosa.

Passo 1: Criar Bactérias

Use o Editor Genético (botão "Criar Bactéria") ou o comando no terminal:

criar RKW

Você pode especificar os genes ou deixar aleatório.

Exemplos válidos:

criar RRR criar WWW criar RKW
Passo 2: Experimentar

Use o comando de cruzamento:

cruzar 1 2 2 1

Onde:

  • 1 2 = IDs das bactérias
  • 2 1 = número de genes a doar (opcional)

Observe os resultados para entender a herança genética.

Passo 3: Testar Resistência

Use o botão "Modo Infecção" ou o comando:

infectar

Isso exporá sua colônia a bacteriófagos. As bactérias sobreviventes serão aquelas com melhor resistência genética.

Analise os padrões de sobrevivência para entender quais combinações genéticas são mais eficazes.

Passo 4: Analisar Resultados

Use as ferramentas do laboratório:

  • Clique nas bactérias na Câmara de Cultivo
  • Use "Examinar Bactéria" para detalhes
  • "Sequenciar Genoma" para análise precisa
  • Consulte o Banco de Dados para estatísticas

O sistema gerará hipóteses automaticamente com base nos dados coletados.

Regras Genéticas

Atenção: Estas são as regras que você precisará descobrir através da experimentação. Leia apenas se quiser spoilers!

Hierarquia de Dominância

Os genes seguem uma hierarquia clara de dominância:

R
Mais dominante
K
Intermediário
W
Mais recessivo

Resistência a Bacteriófagos

A resistência depende da quantidade de genes W na bactéria:

Combinação Genética Exemplo Chance de Sobrevivência
3 genes W WWW 95%
2 genes W RWW 55%
1 gene W RKW 30%
0 genes W RRK 7%

Perguntas Frequentes (FAQ)

Como faço para começar um novo experimento?

Use o botão "Iniciar Experimento" na página inicial ou o comando iniciar no terminal. Isso limpará a câmara de cultivo e permitirá que você comece do zero.

Por que minhas bactérias estão morrendo durante a infecção?

A resistência à infecção depende dos genes W. Tente criar bactérias com mais genes W (como WWW) para aumentar as chances de sobrevivência.

Como posso ver quais genes uma bactéria específica possui?

Clique na bactéria na Câmara de Cultivo ou use o comando analisar [id] no terminal, substituindo [id] pelo número da bactéria.

O que são os eventos aleatórios?

Eventos aleatórios são desafios que podem ocorrer durante seu experimento, como vazamentos de mutagênico ou falhas no sistema de quarentena. Eles adicionam imprevisibilidade ao seu trabalho.

Como desbloqueio conquistas?

Conquistas são desbloqueadas ao realizar descobertas importantes ou completar objetivos específicos, como descobrir a hierarquia de dominância ou criar uma colônia totalmente resistente.

Posso salvar meu progresso?

Sim! Use o comando exportar no terminal para baixar um arquivo com seu progresso atual. Você pode importá-lo depois com o comando importar.
